## Getting Started

### Prerequisites

* node.js
* npm

### Installation & run

1. Clone the repo
   ```sh
   git clone https://github.com/lopezrunco/Currency-exchange-scraper.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Run scraper
   ```sh
   node scraper.js
   ```
All data will be stored in data.json file
